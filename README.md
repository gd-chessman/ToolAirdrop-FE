# Cipher 43 Lab - Landing Page

A modern, responsive landing page for Cipher 43 Lab built with Vue 3, Tailwind CSS, and Vue Router. This project showcases airdrop tools, pricing plans, and detailed airdrop information with a professional dark theme design.


## =ï¿½ Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js**: Version `^20.19.0` or `>=22.12.0` (check with `node --version`)
- **npm**: Usually comes with Node.js (check with `npm --version`)

## =ï¿½ Installation

1. **Clone the repository**

   ```bash
   git clone <repository-url>
   cd cypher_43_landing_page
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

   This will install all required packages including:
   - Vue 3
   - Vue Router
   - Tailwind CSS
   - Vite
   - And other development dependencies

## <ï¿½ Run

To Run the local server with hot-reload:

```bash
npm run dev
```

The application will be available at `http://localhost:5173` (or another port if 5173 is busy).
